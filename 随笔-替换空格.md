####   题目：
替换空格
####  思路：运用查找和替换两个方法；
```
public class Solution {
    public String replaceSpace(StringBuffer str) {
       if(str.length()==0){
       return null;
       }
       int i=0;
        while(str.indexOf(" ",i)!=-1){
            int j=str.indexOf(" ",i);
            str.replace(j,j+1,"%20");
            
            i=j+3;
        }
    }
}
```
