####   题目：
输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。
####  思路：暴力方法是 >>>  32次；   当一个二进制数-1的时候，它最低位的1变成0，如果还有0那全部变成0； 如果它与原值进行与运算，那么原值最低为的1  以及后面的全部变成0；
 

```
public class Solution {
    public int NumberOf1(int n) {
        int count=0;
        while(n!=0){
            count++;
            n=n&(n-1);
        }
        return count;
    }
}
```
